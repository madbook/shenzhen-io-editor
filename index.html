Save file parser

Paste contents of a saved design in the input box, then click "Convert to JSON!" to get the parsed JSON output in the output box. Convert parsed JSON back into the native level format by clicking the "Convert to level format!" button.

more save files can be found in...
  • Windows: Documents\My Games\SHENZEN IO\<Steam account id>
  • Arch Linux: $HOME/.local/share/SHENZHEN IO/<Steam account id>
  • macOS: ~/Library/Application Support/SHENZHEN IO/<Steam account id>
